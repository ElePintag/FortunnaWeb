## 1️⃣ Slides del Banner Principal

**Ubicación en el sitio**: Carrusel de la página de inicio

### Especificaciones Técnicas

| Propiedad | Valor |
|-----------|-------|
| **Ancho** | 1920 píxeles |
| **Alto** | 1080 píxeles |
| **Proporción** | 16:9 (panorámico) |
| **Peso máximo** | 500 KB |
| **Peso recomendado** | 300-400 KB |
| **Formato** | JPG (optimizado) o WebP |
| **Cantidad recomendada** | 3-5 slides |

### Características de Contenido

✅ **Usar**:
- Imágenes horizontales amplias
- Buena iluminación
- Alta resolución
- Colores vibrantes
- Enfoque nítido

❌ **Evitar**:
- Texto pequeño (difícil de leer en móvil)
- Imágenes verticales
- Sobreexposición o subexposición
- Desenfoques
- Marcas de agua grandes

### Herramientas Recomendadas

- **Canva**: Redimensionar a 1920x1080
- **Photoshop**: Exportar para web (calidad 70-80)
- **TinyPNG**: Comprimir sin perder calidad
- **Squoosh**: Convertir a WebP

### Ejemplo de Uso

```
Slide 1: Vista panorámica de la ciudad al atardecer
Slide 2: Interior de una casa moderna (sala amplia)
Slide 3: Terreno con vista a las montañas
```

## 2️⃣ Imagen Principal de Propiedad

**Ubicación en el sitio**:
- Tarjeta en el catálogo
- Encabezado en página de detalle

### Especificaciones Técnicas

| Propiedad | Valor |
|-----------|-------|
| **Ancho** | 1200 píxeles |
| **Alto** | 800 píxeles |
| **Proporción** | 3:2 |
| **Peso máximo** | 300 KB |
| **Peso recomendado** | 200-250 KB |
| **Formato** | JPG o WebP |

### Características de Contenido

✅ **Usar**:
- La mejor vista de la propiedad
- Ángulo que muestre más espacio
- Luz natural preferentemente
- Espacios limpios y ordenados
- Fachada completa o sala principal

❌ **Evitar**:
- Fotos borrosas
- Mala iluminación
- Espacios desordenados
- Ángulos cerrados
- Obstrucciones visuales

### Consejos Fotográficos

1. **Hora ideal**: Entre 10 AM y 4 PM (luz natural)
2. **Altura de cámara**: A nivel de pecho
3. **Posición**: Esquinas de la habitación para captar más espacio
4. **Modo**: Usar modo horizontal (landscape)
5. **Post-procesamiento**: Ajustar brillo y contraste levemente

### Ejemplo de Configuración de Cámara

```
Smartphone:
- Resolución: Alta (12MP+)
- HDR: Activado
- Grid: Activado (regla de tercios)
- Flash: Apagado (usar luz natural)

Cámara DSLR:
- ISO: 400-800
- Apertura: f/8-f/11 (mayor profundidad)
- Velocidad: 1/60s o superior
```

## 3️⃣ Galería de Imágenes (Imágenes Adicionales)

**Ubicación en el sitio**: Galería en página de detalle de propiedad

### Especificaciones Técnicas

| Propiedad | Valor |
|-----------|-------|
| **Ancho** | 1200 píxeles |
| **Alto** | 800 píxeles |
| **Proporción** | 3:2 o 4:3 |
| **Peso máximo** | 250 KB cada una |
| **Peso recomendado** | 150-200 KB |
| **Formato** | JPG o WebP |
| **Cantidad** | 5-10 imágenes |

### Plan de Fotografía Sugerido

**Para Casas**:
1. Fachada exterior
2. Sala principal
3. Cocina
4. Recámara principal
5. Baño principal
6. Jardín o patio (si aplica)
7. Cochera / estacionamiento
8. Vista desde la ventana
9-10. Detalles especiales (chimenea, alberca, etc.)

**Para Terrenos**:
1. Vista panorámica completa
2. Entrada / acceso
3. Diferentes ángulos del terreno
4. Vistas del entorno
5. Servicios disponibles (postes de luz, tomas de agua)
6-8. Perspectivas adicionales
9-10. Alrededores / contexto urbano

### Orden Recomendado

```
Posición 1: Vista más atractiva (hero shot)
Posición 2: Segunda mejor vista
Posiciones 3-7: Espacios importantes
Posiciones 8-10: Detalles y contexto
```

## 📐 Cómo Redimensionar Imágenes

### Método 1: Usando Canva (Online)

```
Pasos:
1. Ir a https://canva.com
2. Click en "Create a design" > "Custom dimensions"
3. Ingresar dimensiones (ej: 1920 x 1080)
4. Subir tu imagen
5. Ajustar para que llene el canvas
6. Download > JPG (calidad 80)
```

### Método 2: Usando Photoshop

```
Pasos:
1. File > Open (abrir imagen)
2. Image > Image Size
3. Cambiar Width: 1920px, Height: 1080px
4. Marcar "Constrain Proportions" si quieres mantener proporción
5. File > Export > Save for Web
6. Formato: JPEG
7. Quality: 70-80
8. Save
```

### Método 3: Usando Herramientas Online Simples

**Resize Image (https://resizeimage.net)**
```
1. Upload imagen
2. Seleccionar dimensiones
3. Download
```

## 🗜️ Cómo Comprimir Imágenes

### Reglas de Compresión

| Formato | Calidad Recomendada | Uso |
|---------|---------------------|-----|
| **JPG** | 70-85 | Fotos con muchos colores |
| **WebP** | 80-90 | Alternativa moderna a JPG |
| **PNG** | Lossless | Logos, transparencias |

### Proceso de Compresión con TinyPNG

```
1. Ir a https://tinypng.com
2. Arrastrar imagen(es)
3. Esperar compresión automática
4. Download (individual o ZIP)

Resultado esperado:
- Reducción: 50-70%
- Pérdida visual: Mínima
- Tiempo: <30 segundos
```

### Conversión a WebP (Squoosh)

```
1. Ir a https://squoosh.app
2. Arrastrar imagen
3. En panel derecho, seleccionar "WebP"
4. Ajustar Quality slider (80-90)
5. Comparar lado a lado
6. Download

Ventajas de WebP:
✓ 25-35% más ligero que JPG
✓ Soporte moderno en navegadores
✓ Calidad superior
```

## ✅ Checklist Pre-Subida

Antes de subir cualquier imagen al panel de administración:

### Verificación Técnica

- [ ] Dimensiones correctas según la sección
- [ ] Peso menor al máximo permitido
- [ ] Formato correcto (JPG, PNG, WebP)
- [ ] Nombre de archivo descriptivo (sin caracteres especiales)
- [ ] Orientación correcta (horizontal para la mayoría)

### Verificación de Calidad

- [ ] Imagen nítida y enfocada
- [ ] Buena iluminación
- [ ] Colores naturales (no sobresaturados)
- [ ] Sin marcas de agua (o discretas)
- [ ] Espacios ordenados (para propiedades)
- [ ] Sin elementos distractores

### Verificación de Contenido

- [ ] Representa fielmente la propiedad
- [ ] No incluye información confidencial
- [ ] No viola derechos de autor
- [ ] Es apropiada para sitio público

## 🎨 Mejores Prácticas Fotográficas

### Iluminación

**✅ Hacer**:
- Fotografiar con luz natural
- Horario: 10 AM - 4 PM
- Abrir cortinas y persianas
- Encender luces interiores
- Evitar contraluz

**❌ Evitar**:
- Flash directo (crea sombras duras)
- Fotografiar al mediodía (luz dura)
- Habitaciones oscuras
- Mezclar luces de distinto color

### Composición

**✅ Hacer**:
- Usar regla de tercios
- Mantener líneas horizontales rectas
- Capturar espacios completos
- Fotografiar desde esquinas
- Incluir puntos de referencia

**❌ Evitar**:
- Ángulos extremos
- Fotos torcidas
- Cortar elementos importantes
- Demasiado cerca (sensación de espacio reducido)
- Demasiado lejos (falta de detalle)

### Preparación del Espacio

**Antes de fotografiar**:
1. Limpiar y ordenar
2. Retirar objetos personales
3. Abrir cortinas
4. Encender luces
5. Arreglar decoración
6. Esconder cables y objetos distractores

## 🔄 Flujo de Trabajo Completo

### Para una Nueva Propiedad

```
1. PLANIFICACIÓN (5 min)
   - Listar espacios a fotografiar
   - Verificar iluminación óptima
   - Preparar equipo

2. PREPARACIÓN (15-30 min)
   - Limpiar espacios
   - Ordenar
   - Iluminación

3. FOTOGRAFÍA (30-45 min)
   - 10-15 fotos mínimo
   - Diferentes ángulos
   - Verticales y horizontales

4. SELECCIÓN (10 min)
   - Elegir las 6-10 mejores
   - Descartar borrosas o mal expuestas

5. EDICIÓN (15-20 min)
   - Redimensionar a 1200x800
   - Ajustar brillo/contraste
   - Comprimir

6. SUBIDA (5 min)
   - Imagen principal primero
   - Galería en orden lógico
   - Verificar en sitio

TIEMPO TOTAL: ~2 horas
```

## ❓ FAQ - Preguntas Frecuentes

### ¿Qué hago si mi imagen es muy pesada?

**R**: Comprímela usando TinyPNG o Squoosh. Si sigue pesada, reduce las dimensiones.

### ¿Puedo usar fotos verticales?

**R**: No recomendado. Los slides y tarjetas están optimizados para horizontal.

### ¿Qué es mejor: JPG o WebP?

**R**: WebP es más eficiente, pero JPG es más compatible. Usa JPG optimizado.

### ¿Cómo tomo fotos profesionales con mi celular?

**R**:
1. Limpia el lente
2. Usa luz natural
3. Activa el grid
4. No uses zoom digital
5. Estabiliza el teléfono

### ¿Cuántas fotos debo subir por propiedad?

**R**: Mínimo 5, ideal 8-10. No más de 15.

### ¿Puedo editar las fotos?

**R**: Sí, pero solo ajustes básicos:
- ✅ Brillo, contraste, nitidez
- ✅ Corregir perspectiva
- ❌ Filtros exagerados
- ❌ Eliminar defectos importantes
- ❌ Cambiar colores drásticamente

### Mi imagen se ve borrosa en el sitio

**R**: Probablemente la dimensión es muy pequeña. Usa las dimensiones recomendadas exactas.
